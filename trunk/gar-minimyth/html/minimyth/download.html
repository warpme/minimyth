Versions

There are multiple MiniMyth distribution versions available for download. In order to decide which one to download, it helps to understand the MiniMyth distribution's version.

The version consists of three parts separated by hyphens (-). The first part is the MythTV version number. The second part is the MiniMyth build system version number. The third part, which may or may not be present, is an extra version. For example, the version 0.21.0-53 means that this MiniMyth distribution contains MythTV 0.21.0 and was built using MiniMyth version 53.

You may have noticed that the MythTV version number and the MiniMyth build system version number are independent. This is done because the MiniMyth build system can build multiple versions of MythTV, depending on the MiniMyth build system configuration settings. For example, the MiniMyth distributions with the versions 0.21.0-53 and trunk.17402-53 would contain different versions of MythTV, but would have been built using the same MiniMyth build system.

Typically, the extra version is used to indicate that the MiniMyth distribution was built using one or more non-default MiniMyth build system configuration settings. Usually, this is done only when providing test builds for troubleshooting. As a result, stable MiniMyth distribution versions rarely have an extra version.

Directories
Latest stable MiniMyth release
A directory containing the latest stable MiniMyth release. The latest stable MiniMyth release is usually the latest stable version of MiniMyth with the latest stable version of MythTV. For the latest stable MiniMyth release with an older stable MythTV release visit the old stable MiniMyth releases directory.
Latest test MiniMyth release
A directory containing the latest test MiniMyth release. There is not always a test MiniMyth release. As result, this directory is sometimes empty.
All stable MiniMyth releases
A directory containing "all" stable MiniMyth releases. In order to preserve server space, not every stable MiniMyth release is kept indefinitely.
All test MiniMyth releases
A directory containing all test MiniMyth releases. When a new stable or test MiniMyth release is added, old test MiniMyth releases are usually deleted. Therefore, there is usually only one test MiniMyth release. However, at times there are multiple test MiniMyth releases because we are testing multiple things (e.g., MiniMyth with different versions of MythTV).
MiniMyth's GAR build system's source files archive
A directory mirroring some of the source code used by MiniMyth's GAR build system. This is done for two reasons. First, this is done to comply with software licenses that require the source to be made available to those whom the binary is made available (e.g. GPL and LGPL). Second, this is done to minimize the impact caused by the scheduled and unscheduled outages of servers (e.g. sourceforge.net). As a result, only a subset of the source code is mirrored here. If you feel that additional source code should be mirrored for either of the above two reasons, then please contact me (< mailto : webmaster at minimyth dot org >) and let me know.
MiniMyth download directory
The directory containing all the MiniMyth download directories.
